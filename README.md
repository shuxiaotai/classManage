### 小学课堂管理手机端

一.登录  

 1.选择身份，老师或者家长，若是未注册的用户会自动注册，然后显示登录界面，若是已经注册的会直接显示登录界面
 
 2.token没过期的话会自动登录
 
 3.登录后的界面发的请求都检测了token是否过期，过期需要重新登录
